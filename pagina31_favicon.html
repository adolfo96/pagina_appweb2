como añadir un favicon a nuestra pagina web

Si se quiere utilizar para todas las páginas del servidor no hace falta utilizar ningún código, basta con colocar el fichero .ico en la carpeta raíz del servidor.

Si por el contrario se quiere utilizar el icono en una sola página, ir variando utilizando diferentes iconos o bien si el icono no se llama 'favicon.ico' o está ubicado en otra carpeta se debe añadir un código dentro del 

  
  <link  rel="icon"   href="D:\mincho\html\imagenes/favicon.png" type="image/png" />
